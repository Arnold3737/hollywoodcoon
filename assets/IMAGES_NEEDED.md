# Images Needed for Production

> These are placeholder image files referenced in the HTML but not yet present in `assets/`.
> Each responsive image requires 7 variants: three AVIF sizes, three WebP sizes, and one full-size JPG fallback.
> The source `.webp` files in this directory can be used to generate all variants.
